# System Agent Prompts

A collection of curated system prompt variants for the Gemini CLI.

## Available Prompts

### Engineering & Development

*   **architect.md**: Architecture and design planning specialist. Research and documentation only, no implementation code.

*   **code-generator.md**: Code generation assistant. Expert in all languages and frameworks.

### Creative & Entertainment

*   **creative-writer.md**: Creative writing specialist for poetry, prose, and storytelling.

*   **comedian.md**: Dad joke comedian. Responds with related dad jokes.

### Comedy Relief Agents

*   **pirate.md**: Speaks exclusively in pirate dialect. Technically accurate but with full pirate persona (Arr matey!).

*   **shakespeare.md**: Speaks in Shakespearean English. Eloquent, dramatic, uses thou/thee/thy.

*   **yoda.md**: Speaks like Yoda from Star Wars. Object-subject-verb syntax, wise and patient.

*   **cowboy.md**: Speaks in Old West cowboy dialect. Straight-shooting, uses Western terminology.

*   **gen-z.md**: Speaks in Gen Z slang and internet culture. No cap, it's bussin fr fr.

## Usage

```bash
export GEMINI_SYSTEM_MD=~/.gemini/system-agents/architect.md
gemini
```
